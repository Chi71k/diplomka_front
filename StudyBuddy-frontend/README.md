# StudyBuddy Frontend

Фронтенд приложения StudyBuddy (React + Vite). Регистрация, вход, профиль, курсы.

## Как запустить

### 1. Установить зависимости

Из папки `project-front`:

```bash
npm install
```

### 2. Запустить dev-сервер

```bash
npm run dev
```

Приложение откроется по адресу **http://localhost:5173** (или другой порт, если 5173 занят — смотри вывод в терминале).

### 3. Бэкенд (обязательно для работы приложения)

Фронт ходит к API через прокси Vite. Нужно запустить три сервиса бэкенда на портах:

- **8080** — Auth (логин, регистрация)
- **8081** — Users (профиль)
- **8082** — Courses (курсы)

И поднять PostgreSQL (например, `docker compose up -d` из корня бэкенда). Без запущенного бэкенда логин и загрузка данных не будут работать.

## Скрипты

| Команда | Описание |
|--------|----------|
| `npm run dev` | Запуск dev-сервера с hot reload |
| `npm run build` | Сборка для продакшена в папку `dist/` |
| `npm run preview` | Локальный просмотр собранной версии |
| `npm run lint` | Проверка кода ESLint |

## Переменные окружения

Опционально в корне `project-front` можно создать `.env`:

- `VITE_API_BASE` — базовый URL API (по умолчанию пусто: запросы идут через прокси на localhost).

Прокси настроен в `vite.config.js`: запросы к `/api/v1/auth` идут на 8080, к `/api/v1/users` — на 8081, к `/api/v1/courses` — на 8082.
